  Проект называется "Система управления заказами на домашние работы (сантехники, столяры, строители)".

  Этот проект я делал в процессе обучения на курсах ПВТ. 

  Проект разделен на три модуля, модуль Бек-энд (ДАО), модуль Бизнес-логики (сервисы) и Фронт-энд.

  В модуле Бек-энда использовал Hibernate+Spring, бизнес-модуль - Spring, веб-модуль - Spring MVC+
Spring-security+Tiles+Bootstrap.


  Данные хранятся в БД Oracle XE 11g (в проекте в папке sql лежит dump базы и скрипты для экспорта/импорта) 

  Реализованный функционал:

     Пользователи приложения разделены на две группы, администраторы(ROLE_ADMIN) и пользователи(ROLE_USER).
      
     Администраторы могут просматривать список рабочих, добавлять и удалять их.
	
     Пользователи могут видеть список уже сделанных заказов, а так же добавлять новые заказы.
